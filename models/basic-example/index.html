TuringModels

Basic Example

We define a simple Gaussian model with unknown mean and variance.

  1. Model
  2. Output

Model

using Turing

@model function gdemo(x, y)
  s ~ InverseGamma(2, 3)
  m ~ Normal(0, sqrt(s))
  x ~ Normal(m, sqrt(s))
  y ~ Normal(m, sqrt(s))
end;

Output
